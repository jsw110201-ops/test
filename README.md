# 🎮 Tetris with Effects

## 소개
`Tetris with Effects`는 HTML, CSS, JavaScript로 구현된 웹 기반 테트리스 게임입니다.  
단순한 테트리스가 아닌, 점수·레벨·콤보 시스템과 화면 흔들림 효과, 사운드, 점수 시각화 등의 **다양한 게임 요소**가 포함되어 있습니다.  

---

## 주요 기능

- **점수 시스템**
  - 한 줄 제거 시 점수 증가
  - 콤보 점수 적용 (연속 줄 제거 시 추가 점수)
  - 레벨에 따라 점수 가중치 증가

- **레벨 시스템**
  - 제거한 줄 수에 따라 레벨 상승
  - 레벨이 높아질수록 블록 낙하 속도 증가

- **콤보 시스템**
  - 연속 줄 제거 시 콤보 점수 추가
  - 점수 시각화로 +점수 표시

- **화면 효과**
  - 줄 제거 시 화면 흔들림(Shake) 효과로 타격감 제공

- **사운드 효과**
  - 블록 이동, 회전, 줄 제거, 게임오버에 따른 효과음 적용

- **최고 기록 저장**
  - LocalStorage를 이용해 브라우저별 최고 기록 유지

- **게임 조작**
  - 화살표 키: 좌우 이동 / 아래: 한 칸 내려가기 / 위: 회전
  - Space: 하드 드롭(즉시 떨어뜨리기)

---

## 설치 및 실행

1. HTML 파일을 다운로드합니다.
2. 웹 브라우저에서 파일을 열어 게임을 플레이할 수 있습니다.
3. 서버 없이도 로컬에서 실행 가능하며, PC/태블릿에서도 호환됩니다.

---

## 기술 스택

- **HTML5**: 게임 구조 및 UI
- **CSS3**: 디자인 및 효과
- **JavaScript (ES6)**: 게임 로직, 점수 시스템, 화면 효과
- **Canvas API**: 블록 렌더링, 화면 흔들림 효과
- **LocalStorage**: 최고 기록 저장

---

## 게임 규칙

1. 블록을 회전하고 좌우로 이동하여 한 줄을 채워 제거합니다.
2. 한 줄 제거 시 점수가 증가하며, 연속 제거 시 콤보 점수 적용
3. 레벨이 올라갈수록 블록 낙하 속도가 빨라집니다.
4. 블록이 맨 위까지 쌓이면 게임 오버입니다.
